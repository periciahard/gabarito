# Leitor de Cartão-Resposta

Site estático para leitura de cartão-resposta individual com 26 questões e alternativas A-E.

## Como publicar no GitHub Pages

1. Crie um repositório no GitHub.
2. Envie os arquivos deste pacote para a raiz do repositório.
3. No GitHub, vá em **Settings > Pages**.
4. Em **Build and deployment**, selecione:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
5. Salve e aguarde o link do GitHub Pages.

## Arquivos

- `index.html`: site completo.
- `.nojekyll`: evita processamento pelo Jekyll.
- `README.md`: instruções.

## Uso

1. Abra o site no navegador.
2. Envie uma foto do cartão-resposta.
3. Clique em **Ler cartão-resposta**.
4. Confira as respostas detectadas.
5. Insira o gabarito oficial, se quiser corrigir automaticamente.
6. Exporte o resultado em CSV.

## Observações

A leitura funciona melhor com foto nítida, boa iluminação, cartão sem muita inclinação e bolhas bem preenchidas.
