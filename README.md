# Leitor de Cartão-Resposta — V4 Mobile

Versão otimizada para celular e GitHub Pages.

## Recursos

- Leitura de 52 questões.
- Q1 a Q26: Português.
- Q27 a Q52: Matemática.
- Campo manual para nome do aluno.
- Câmera traseira no celular.
- Guia visual para enquadrar os 4 quadrados pretos.
- Botões maiores e barra inferior para uso em campo.
- Exportação em CSV e Excel simples (.xls).
- Salvamento local do último resultado.

## Publicação no GitHub Pages

Envie todos os arquivos deste pacote para o repositório e ative Pages em Settings > Pages.
